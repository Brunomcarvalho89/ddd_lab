# DDD_lab

Implementação do contexto de Identificação e Autenticação de usuários usando o conhecimentos de Domain Driven Design em Java.

Projeto AuthenticationIdentity é o projeto principal com api para cadastro de usurios e geração do token de acesso as funcionalidades.
Projeto EmailSender responsável por notificar via email.
Projeto ResowCommom responsável por conter as classes bases utilizadas nos outros projetos.
